# IM —— 一个简单的 C 语言即时通讯项目

这是使用 **C 语言 + Socket + SQLite** 编写的一个简单即时通讯（IM）项目。

项目整体分为 **Client（客户端）**、**Server（服务端）** 和 **IM（公共库）** 三部分。

---

## ✨ 功能简介

目前实现或计划实现的功能包括：

- ✅ 用户注册 / 登录 / 登出  
- ✅ 好友添加 / 删除 / 查询  
- ✅ 私聊聊天  
- ✅ 基于 SQLite 的数据存储  
- ✅ 自定义 IM 协议  
- ✅ Linux / Windows 跨平台支持

---
